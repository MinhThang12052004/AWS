# **WEEK 8 WORKLOG**

### **Week 8 Objectives**

* Learn and configure **AWS RDS** (Relational Database Service) for MySQL and PostgreSQL.
* Master **Amazon DynamoDB** for NoSQL databases and perform CRUD operations.
* Practice **S3 Advanced** features: versioning, lifecycle policies, replication.
* Implement database security with encryption at rest and in transit.
* Design multi-AZ and read replica architectures for high availability.
